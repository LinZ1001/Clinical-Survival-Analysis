# Clinical Survival Analysis of Lung Cancer Patients

## Overview
This project performs survival analysis on lung cancer patient data to identify factors associated with mortality risk.

The analysis demonstrates common statistical techniques used in clinical data science, including Kaplan–Meier survival curves and Cox proportional hazards regression.

## Dataset
The dataset comes from the R `survival` package and contains clinical data from 228 lung cancer patients.

Key variables include:
- survival time
- patient age
- sex
- ECOG performance score

## Research Question
Which patient characteristics influence survival outcomes in lung cancer patients?

## Methods
The analysis includes:

1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Kaplan–Meier survival curves
4. Cox proportional hazards regression

## Tools Used
- R
- survival package
- survminer
- tidyverse
- ggplot2
